## HTML - Dasar

Penulisan website harus diawali dan diakhiri dengan tag <code><html> ... </html><code>
<code><html> ... </html><code>
  
Struktur penulisan html :

<code>
  <DOCTYPE!>
    <html>
      <head>
        <title> ini judul web </title>
      </head>
      <body>
        <p>...</p>
      </body>
    </html>
    </code>
  
  Dalam file html terdapat <code><head>...</head></code> dan <code><body>...</body></code>.
