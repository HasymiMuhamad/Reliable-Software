## HTML - Dasar

Penulisan website harus diawali dan diakhiri dengan tag ```<html> ... </html>```
```<html> ... </html>```
  
Struktur penulisan html :

```
  <DOCTYPE!>
    <html>
      <head>
        <title> ini judul web </title>
      </head>
      <body>
        <p>...</p>
      </body>
    </html>
   ```
  
  Dalam file html terdapat ```<head>...</head>``` dan ```<body>...</body>```.
